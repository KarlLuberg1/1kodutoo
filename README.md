Funktsionaalsus:
Selles kella rakenduses on võimalik ekraanil klikates liikuda 3 erineva taustapildi vahel.
Lisaks saab klaviatuuri klahve vajutades vahetada kella ja kuupäeva teksti värvi.

Lehe päisesse olen lisanud nupud:
* Keele vahetamise nupp võimaldab eesti ja inglise keele vahel vahetada.
* Timeri nupuga saab käima panna timeri, kus süsteem küsib sinult sekundites timeri pikkust.
* Font nupp võimaldab kella ja kuupäeva fonti muuta liikudes viie erineva fonti vahel.
* Noodi ikooniga nuppu vajutades mängib lühike heliklipp.
